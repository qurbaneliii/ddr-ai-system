# Assumptions and unresolved terms

- **SoR:** not defined in supplied files. Stored verbatim and not expanded.
- **Pressure-time unit:** the y-axis states `Pressure` without a unit. Stored as unknown.
- **Identity mappings:** no mapping is assumed among 14 DDR wellbores, 30 profile identifiers, 30 pressure-time filenames, or the four repeated displayed series.
- **Campaign gaps:** date gaps do not imply missing reports; trends exclude the known period-before-spud case by default and do not interpolate gaps.
- **Anomalies:** all automated detections are candidates until human/domain validation.
- **OCR availability:** a plot can retain calibrated pixel evidence and an explicit numeric-calibration warning when local OCR cannot establish tick values.
